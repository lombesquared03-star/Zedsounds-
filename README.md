# # ZedSounds 🇿🇲
A lightweight platform for streaming and downloading Zambian music.

## Current Features
- 🎵 Built-in Audio Player
- 📤 Local Upload Simulation
- 🇿🇲 Zambian Genre Categories (Zed-Pop, Kalindula, Zamrock)
- ✨ Gemini AI Artist Scout (Find new artists instantly)
